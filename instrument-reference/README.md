# instrument-reference

Reference material for the instruments used across the study: manuals, acquisition
software, wiring maps, and calibration notes. One subfolder per instrument.

| Subfolder | Instrument | Used by |
|-----------|-----------|---------|
| `thread-tensometer/` | OMEGA DFG51-2 digital force gauge and its LabVIEW logger | thread-strength |

These are reference documents, not analysis inputs, so they are kept out of any single
analysis folder. Add a new subfolder here when another instrument's documentation is added.

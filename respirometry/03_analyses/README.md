# 03_analyses

Generated outputs, under `metabolic-rate/` (both timepoint scripts write here).

| Subfolder | Produced by | Contents |
|-----------|-------------|----------|
| `metabolic-rate/QC_plots/<date>/` | either script | Oxygen-vs-time plots with fitted slope, one per run, for QC |
| `metabolic-rate/output/` | either script | `SMR_slope_<date>.xlsx`, `SMR_slope_final.xlsx`, `processed_summary.xlsx` |

`processed_summary.xlsx` is a combined summary not written by the current scripts; it is
retained here for provenance.
